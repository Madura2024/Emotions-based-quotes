# Emotion-Based Quote Generator

A premium, student-focused web application that delivers personalized quotes based on your current mood (Stress, Failure, or Motivation).

## Features
- **Mood Selection**: Choose from 3 distinct emotional states.
- **Curated Quotes**: Receive wisdom specifically tailored to help students cope or get inspired.
- **Premium UI**: Built with a glassmorphism aesthetic, rich animated gradients, and smooth interactions.
- **Responsive Design**: Looks great on both desktop and mobile devices.

## Tech Stack
- React
- Vite
- CSS (Custom Glassmorphism & Animations)

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Madura2024/Emotions-based-quotes.git
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run the development server**
   ```bash
   npm run dev
   ```
